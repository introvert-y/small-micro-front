# small-micro-front

精简的single-spa例子，基于system.js加载

## 安装app1依赖

cd app1 && cnpm install/yarn

## 启动root-config

npm run serve

## 安装app2依赖

cd root-config && cnpm install/yarn

## 启动app2 serve

模拟实际调用场景 <br/>

npm run build<br/>

node serve.js<br/>

## 安装root-config依赖

cd root-config && cnpm install/yarn

## 启动root-config

npm run start
